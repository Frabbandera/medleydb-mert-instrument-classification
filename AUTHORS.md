# Authors

This repository documents a team project developed for the Selected Topics in Music and Acoustic Engineering course at Politecnico di Milano, Academic Year 2025-2026.

Project authors:

- Francesco Bandera
- Anna Impembo
- Filippo Longhi
- Eleonora Serra

The repository is presented as an academic portfolio and research-code case study. Contributions were developed collaboratively across dataset preparation, modelling, experiments, validation and reporting.
