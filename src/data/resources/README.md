# Bundled MedleyDB release manifests

These two small text files contain the MedleyDB V1 and V2 track identifiers. They
let the indexer report release coverage without requiring a checkout of the
legacy MedleyDB Python repository.

The identifiers were reproduced from the `marl/medleydb` project resources
(`tracklist_v1.txt` and `tracklist_v2.txt`). They contain names only: no audio,
annotations, library code, or documentation. The upstream project is distributed
under the MIT License. MedleyDB audio remains a separate dataset and is never
included in this repository.
